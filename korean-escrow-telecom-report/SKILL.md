# 🛒 에스크로 발급 워크어라운드 & 통신판매업 신고 상세 가이드

## 📌 1. 네이버 스마트스토어 에스크로 확인증 즉시 발급 (닭과 계란 해결)

PG사나 은행에서 에스크로(구매안전서비스 이용확인증)를 발급받으려면 통신판매업 신고번호가 필요한데, 정부24에서 통신판매업 신고를 하려면 에스크로 확인증이 필요한 무한 루프(닭과 계란 문제)가 발생합니다.

이를 **비용 0원에 즉시 해결하는 스마트스토어 워크어라운드**입니다.

### 상세 단계 및 화면 입력 항목
1. **네이버 스마트스토어 센터 (`sell.smartstore.naver.com`) 접속**:
   * `가입하기` ➔ 가입 유형: **[사업자]** 선택.
2. **사업자등록번호 입력**:
   * 홈택스에서 발급받은 사업자등록번호 10자리 입력 ➔ 인증.
3. **통신판매업 신고 여부 선택 (핵심)**:
   * **입력 칸**: 통신판매업 신고 여부 선택란
   * **선택**: **[미신고]** 선택 (이유: 아직 신고 전이므로 미신고로 가입 가능).
4. **판매자 정보 및 주소 입력**:
   * 상호명, 대표자명, 사업장 주소지 입력.
   * **개인정보 보호 팁**: 스마트스토어 상품을 등록해서 실제 판매 개시를 하지 않는 한, 대표님의 개인 핸드폰 번호나 주소는 외부에 노출되거나 검색되지 않으므로 안심하고 본인 번호 입력.
5. **서류 제출 (주민번호 뒷자리 마스킹 필수!)**:
   * 사업자등록증명원 또는 사업자등록증 첨부.
   * **🚨 마스킹 규칙**: 주민등록번호 뒷자리 7자리(`930301-1*******`)를 그림판이나 스마트폰 편집기/PDF 편집기로 검은색으로 완전히 가린 후 이미지(JPG/PNG) 형태로 업로드.
6. **구매안전서비스 이용확인증 다운로드**:
   * 심사 승인(보통 당일~1일) 후 스마트스토어 관리자 로그인 ➔ `판매자정보` ➔ `판매자 정보` 메뉴 접속 ➔ 우측 상단 `[구매안전서비스 이용확인증]` 버튼 클릭 ➔ **PDF 다운로드/저장**.

## 📋 2. 정부24 (`gov.kr`) 통신판매업 신고 상세 칸 채우기

다운로드받은 에스크로 확인증 PDF를 들고 정부24에서 온라인으로 신고하는 상세 과정입니다.

### [신고 화면 1] 신청인 정보
1. **대표자 성명 / 생년월일 / 연락처 / 이메일**: 로그인 정보 기반 자동 입력 확인.
2. **주소 (주민등록상 주소)**: 본인 주민등록지 주소 입력.

### [신고 화면 2] 사업자 등록 사항 (회사 정보)
1. **상호 (법인명)**: `에이아잉` (사업자등록증 상 상호)
2. **사업자등록번호**: `102-36-54285` (10자리 숫자)
3. **사업장 소재지 (전화번호)**:
   * 사업자등록증에 기재된 사업장 주소 입력. (자택 주소)

### [신고 화면 3] 구체적 판매 정보 (업종 및 수단)
1. **판매방식**: 
   * **체크박스**: **[인터넷]** 체크.
2. **취급품목**:
   * **체크박스**: **[서비스/강의]**, **[디지털콘텐츠]**, **[의류/잡화]** 등 본인이 다루는 품목 체크.
3. **인터넷 도메인 이름**:
   * **입력 칸**: `인터넷 도메인` 입력란
   * **가이드**: 본인 웹사이트 주소 (`ai-ing.org`) 또는 스마트스토어 주소 입력.
4. **호스팅업체 명칭**:
   * **입력 칸**: `호스팅업체` 입력란
   * **가이드**: `Cloudflare`, `네이버`, `카페24` 등 본인이 사용하는 호스팅사 입력.

### [신고 화면 4] 구매안전서비스(에스크로) 서류 첨부
1. **선구비서류 첨부**:
   * **파일 업로드**: 아까 다운로드받은 **네이버 스마트스토어 구매안전서비스 이용확인증 PDF** 첨부.
2. **수령 방법 선택**:
   * `온라인발급(전자문서지갑)` 또는 `온라인발급(본인출력)` 선택.

### [신고 화면 5] 제출 및 면허세 납부
1. **민원 신청하기** 클릭 ➔ 1~2일 후 처리 완료 문자 수신.
2. 위택스(`wetax.go.kr`) 또는 문자 안내에 따라 **등록면허세(연 약 40,500원 내외)** 납부.
3. 통신판매업 신고증 발급 완료 ➔ **신고번호(`제 2026-서울서초-2131호`) 획득**.
