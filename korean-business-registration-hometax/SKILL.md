# 📝 Step 1. 국세청 홈택스 사업자등록 메뉴 경로 & 화면별 입력 칸(Field) 극상세 가이드

이 가이드는 실물 재고나 오프라인 매장 없이 디지털 서비스, 온라인 강의, 코칭, 소프트웨어를 판매하는 **무재고 디지털 통신판매업(간이과세자)** 신청을 위한 국세청 홈택스 실제 접속 경로 및 화면별 입력 칸 채우기 가이드입니다.

---

## 🌐 1. 홈택스 접속 및 메뉴 클릭 경로 (Step-by-Step)

1. **국세청 홈택스 공식 사이트 접속**: `https://www.hometax.go.kr`
2. **인증 로그인**: 
   * 우측 상단 `[로그인]` 클릭 ➔ `간편인증(카카오톡, 토스, 네이버 등)` 또는 `공동·금융인증서`로 대표자 본인 인증 로그인.
3. **사업자등록 메뉴 이동 경로**:
   * 상단 네비게이션 메인 메뉴 중 **`[국세증명·사업자등록·세금관련 신청/신고]`** 마우스 오버.
   * 드롭다운 메뉴 중 **`[사업자등록 신청·정정·자진발급]`** 항목 확인.
   * 세부 메뉴에서 **`[개인 사업자등록 신청]`** 글자를 클릭!
   * *(※ 빠르게 이동하는 법: 홈택스 메인 검색창에 `개인사업자등록` 검색 후 검색결과에서 바로 이동)*

---

## 📋 2. 신청 화면별 실제 버튼 & 입력 칸(Field) 하나하나 채우기

신청 페이지 진입 후 화면 순서대로 채워야 하는 실제 입력 항목입니다.

### [화면 1] 인적사항 입력 (기본 인적 정보)
* **`상호명(상호)` [필수 입력란]**:
  * 브랜딩할 사업자 이름 입력 (예: `{상호명}` 한글/영문 가능).
* **`주민등록번호` / `성명`**: 로그인 정보로 자동 입력되어 있음.
* **`사업장 전화번호` / `휴대전화번호` [필수 입력란]**:
  * 대표자 본인 휴대폰 번호 입력.
* **`이메일 주소` [필수 입력란]**:
  * 전자세금계산서 및 국세청 모바일/이메일 알림을 수신할 이메일 입력.
* **`국세환급금 계좌신청` [선택 입력란]**:
  * 환급금 발생 시 입금받을 은행 및 계좌번호 선택 입력.

### [화면 2] 사업장 소재지 (주소지)
* **`주소지 본인 소유 여부` [선택 라디오 버튼]**:
  * 자택이 본인 명의인 경우 ➔ `본인소유` 체크
  * 자택이 부모님/타인 명의인 경우 ➔ `타인소유` 체크 (※ 무상 거주 시 **[무상사용승낙서]** 첨부 필요)
  * 임대차(월세/전세 또는 비상주 오피스)인 경우 ➔ `타인소유` 체크 (※ 임대차계약서 첨부 필요)
* **`주소 검색` [버튼 클릭]**:
  * 팝업창에서 무재고 디지털 통신판매업을 운영할 자택 또는 비상주 사무실 도로명 주소 입력 ➔ `주소지 적용`.

### [화면 3] 업종 선택 (주업종 & 부업종 등록)
* **`업종 입력/수정` [버튼 클릭]** ➔ 업종 선택 팝업창 열림.
1. **주업종 (전자상거래 소매업) 등록**:
   * `업종코드` 입력란에 **`525101`** 입력 후 `[조회]` 클릭.
   * 검색 결과 목록에서 **`전자상거래 소매업`** (한국표준산업분류 `47911`) 선택 ➔ `[등록]` 클릭.
2. **부업종 (교육지원 서비스업 등) 등록**:
   * `[업종 추가]` 버튼 클릭.
   * `업종코드` 입력란에 **`809009`** (기타 교육지원 서비스업 - 개인과외, 멘토링, 인강용) 입력 후 `[조회]` ➔ 선택 ➔ `[등록]` 클릭.
   * (필요 시 `749609` 경영컨설팅업 추가 등록 가능).
* **`[업종 등록 완성]`** 버튼을 눌러 팝업창 닫기.

### [화면 4] 사업자 유형 및 개업일자 입력
* **`개업일자` [날짜 선택란]**:
  * 달력 아이콘 클릭 ➔ 신청하는 당일 날짜 선택.
* **`과세 유형 선택` [라디오 버튼 - 가장 중요!]**:
  * **`[간이]` 라디오 버튼 체크!**
  * *(※ 연 매출 8,000만 원 미만 예상 시 부가세 감면 혜택 극대화. 부가세 납부 면제 기준 4,800만 원)*
* **`종업원 수` [입력란]**: `0` 명 입력.
* **`자기자금` / `타인자금` [입력란]**: `0` 원 또는 소액 입력.

### [화면 5] 서류 첨부 및 최종 제출
* **`제출서류 선택` [파일 업로드]**:
  * 자택 무상 거주 시 ➔ `무상사용승낙서` (PDF/JPG) 업로드
  * 임대차 시 ➔ `임대차계약서 사본` 업로드
* **`[신청서 제출하기]`** 버튼 클릭!
* **처리 기간**: 보통 1~2일 내 관할 세무서 담당자 승인 후 완료 문자가 수신됩니다.
