# 📝 홈택스 개인사업자등록 신청 상세 칸 채우기 가이드

## 📌 1. 간이과세 가능 업종 vs 배제 업종 (사전 체크)

개인사업자 신청 시 `간이과세자`를 선택하더라도, 특정 업종이나 지역은 국세청 법령에 의해 **간이과세가 전면 배제(불가)**되고 무조건 일반과세자로 자동 전환됩니다.

### ⭕ 간이과세 적용 가능 주요 업종 (1인 창업 추천)
* **전자상거래 소매업 (업종코드 `525101`)**: 온라인 쇼핑몰, 인강 판매, 디지털 콘텐츠 판매, 정보 제공 서비스 등.
* **기타 교육지원 서비스업 (업종코드 `809009`)**: 1:1 개인 과외, 코딩 멘토링, 특강, 교육 프로그램 개발.
* **컴퓨터 학원 / 기타 서비스업**: 비대면 및 오프라인 소규모 교육.
* **소매업, 음식점업, 숙박업** (일부 지역 제외).

### ❌ 간이과세 전면 배제 업종 (일반과세만 가능)
* **제조업**: (단, 과자점, 양복점, 양장점 등 최종소비자를 상대하는 일부 사업은 제외)
* **도매업 및 상품중개업**: (소매와 달리 B2B 도매는 간이 불가)
* **부동산매매업 및 부동산임대업**: (특정 광역시 이상 지역의 임대사업)
* **전문‧기술‧서비스업**: 변호사, 변리사, 법무사, 공인회계사, 세무사, 의사, 약사, 건축사, 감정평가사 등 전문 직종.
* **광업, 금융‧보험업**: 전문 금융업.
* **과세유흥장소**: 유흥주점 등.

## 📋 2. 홈택스 신청 화면별 실제 입력 칸(Field) 상세 안내

홈택스(`hometax.go.kr`) ➔ `국세증명·사업자등록·세금관련 신청/신고` ➔ `개인사업자등록 신청` 화면 진입 시 채워야 하는 실제 입력 항목입니다.

### [화면 1] 인적사항 입력 (기본 정보)
1. **상호명 (상호)**: 
   * **입력 칸**: `상호명` 입력란
   * **가이드**: 브랜딩할 사업자 이름을 입력 (예: `에이아잉`). 한글/영문 가능.
2. **주민등록번호 / 성명**: 로그인 정보로 자동 입력됨.
3. **사업장 전화번호 / 휴대전화번호 / 이메일**:
   * **입력 칸**: 각 연락처 입력란
   * **가이드**: 실제 연락받을 본인 핸드폰 번호와 이메일 입력. (국세청 알림 수신 동의 체크)

### [화면 2] 사업장 소재지 (주소지)
1. **기본주소 및 상세주소**:
   * **입력 칸**: `주소 검색` 버튼 클릭 ➔ 도로명 주소 입력
   * **가이드**: 실물 매장 없이 디지털 서비스/통신판매를 하는 경우 **자택 주소지** 입력 가능.
2. **자진발급 여부 / 주소지 본인 소유 여부**:
   * **본인 소유 자택**: `본인소유` 체크 (별도 서류 불필요)
   * **타인/부모님 소유 자택**: `타인소유` 체크 ➔ 무상 거주 시 **[무상사용승낙서]** PDF/JPG 첨부
   * **임대차 자택**: `타인소유` 체크 ➔ 임대차계약서 임대인/임차인 정보 및 보증금/월세 금액 입력 후 계약서 첨부.

### [화면 3] 업종 선택 (주업종 & 부업종)
1. **주업종 등록**:
   * `업종 입력/수정` 버튼 클릭 ➔ 업종코드 검색 ➔ `525101` 입력 ➔ `전자상거래 소매업` 선택.
   * **산업분류코드**: `47911` (통신판매업) 자동 연동.
2. **부업종 등록**:
   * 다시 `업종 추가` 클릭 ➔ 업종코드 `809009` 검색 ➔ `기타 교육지원 서비스업` 선택.
   * (필요시 `749609` 경영컨설팅업 추가 가능).

### [화면 4] 사업자 유형 및 개업일자 입력
1. **개업일자**:
   * **입력 칸**: `개업일자` 날짜 선택란
   * **가이드**: 신청하는 당일 날짜 또는 가까운 미래/과거 날짜 선택.
2. **과세 유형 선택**:
   * **입력 칸**: `간이` / `일반` / `면세` 라디오 버튼
   * **가이드**: **[간이]** 선택 (연 매출 8,000만 원 미만 예상 시 혜택 극대화).
3. **창업자금 / 종업원 수**:
   * 종업원 수: `0` 명 (1인 사업자)
   * 자기자금 / 타인자금: `0` 원 또는 소액 입력.

### [화면 5] 서류 첨부 및 최종 제출
1. **첨부 서류 선택**:
   * 자택 무상 거주 시: `무상사용승낙서` 첨부
   * 임대차 시: `임대차계약서 사본` 첨부
2. **확인 및 신청 제출**: `신청서 제출하기` 클릭 ➔ 관할 세무서 담당자 확인 후 1~2일 내 발급 완료.
